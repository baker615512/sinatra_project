This app provides people a place to share their favorite board games. Anyone can see the list of games that have been recommended, but it is necessary to sign up in order to add your own favorite games. You will only be able to edit or delete the games you have shared.

Installing this app is fairly simple. First, fork and clone the repo. You will want to run bundle install in your console. After that, you will be able to run shotgun in your console. Everything should be ready for you to open your browser and navigate to localhost:9393
